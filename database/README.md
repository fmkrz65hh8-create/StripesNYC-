# STRIPESNYC Database

This folder contains the database planning files for STRIPESNYC.

## Files

- `schema.sql` — database table structure
- `seed.sql` — sample starter data

## Core Tables

- `officials`
- `assignors`
- `games`
- `assignments`
- `availability`
- `crewsync_messages`

## Purpose

The database will support:

- Official profiles
- Assignor profiles
- Game schedules
- Crew assignments
- Availability tracking
- CrewSync Live™ communication

## MVP Priority

The first database version should support:

1. Creating official profiles
2. Creating games
3. Assigning officials to games
4. Tracking availability
5. Sending CrewSync Live™ messages
